# HR Attrition Analysis Dashboard

## Project Overview

This project analyzes employee attrition data to identify key factors affecting workforce retention. The dashboard provides insights into employee turnover trends across salary levels, age groups, job roles, and other demographic factors.

##  Tools & Technologies

* Power BI
* Excel

##  Key Features

* KPI metrics including total employees, attrition count, and attrition rate
* Attrition analysis by age group, salary slab, job role, and education
* Interactive dashboard with filters for department-level insights
* Visual representation of workforce trends and employee distribution

##  Key Insights

* Highest attrition observed among employees in the 26–35 age group
* Employees in the lower salary b (≤5K) show significantly higher attrition
* Job roles such as Laboratory Technician and Sales Executive contribute most to attrition
* Attrition trends vary based on experience and education levels

## 🚀 Business Impact

This dashboard helps HR teams identify high-risk employee segments and supports data-driven decisions to improve retention strategies.
